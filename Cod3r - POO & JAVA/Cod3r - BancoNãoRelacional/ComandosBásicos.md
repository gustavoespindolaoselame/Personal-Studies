0 - Start MongoDb:
        sudo systemctl start mongod

1 - Stop MongoDb:
        sudo systemctl stop mongod

2 - Restart MongoDb:
        sudo systemctl restart mongod

3 - Verify MongoDb:
        sudo systemctl status mongod

4 - Begin MongoDb:
        mongosh

5 - Show dbs:
        show dbs

6 - Show Collections:
        show collections

7 - Use db:
        use dbName

8 - Create Collection:
        db.createCollection('')
